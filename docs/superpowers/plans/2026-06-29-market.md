# engine `market` 模块 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: 用 ultracode workflow 串行 TDD（红→绿→提交），与 money/config/orderbook/account 同款，一路跑到全绿 + 独立验证门。Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现 engine 的 `market` 模块（ADR-0005 §3）—— 单只股票的市场状态层：包装 `OrderBook`，叠加涨跌停拒单、last_price 记录、隐藏公允价 V 几何均值回复演化、日终重置。

**Architecture:** `Market` 持 `book: OrderBook` + `last_price/last_close/fundamental_value(V)/limit_pct`。`place` 先涨跌停校验（超限拒单）再委托 `book.place`，末笔成交更新 last_price，返回 MatchResult（不碰 account）。`evolve_v` 几何均值回复（种子化 RNG，纯整数银行家舍入）。复用 orderbook/money/strategy.Rng/account.StockCode。

**Tech Stack:** Rust 2021，`serde`/`serde_json`/`thiserror`（workspace 已就位）。测试 `cargo test -p engine`。

## Global Constraints

- **铁律二（防御式）：** 涨跌停超限 → `MarketError::LimitExceeded` 拒单（不静默 clamp）；V 参数非法 / V 跨零 → `Err`；book/money 错误透传（`#[from]`）；**禁止**静默存非正 V、禁止吞错误、禁止裸 unwrap 业务路径。
- **铁律一（TDD）：** 每任务严格 红→绿→提交；先写失败测试再写实现。
- 价格/V 全程 `Money`（i64 分）；**禁止 f64 存储权威状态**。f64 仅在 V 演化的「乘 drift」边界出现，立即经 `round_half_to_even_i64`（market 自带，纯整数）落回整数分。
- 依赖只用 workspace 已声明的 `serde`/`serde_json`/`thiserror` + std；**不引入新依赖**。
- 复用：`engine::orderbook::{OrderBook, Order, Trade, Side, OrderId, OrderError, MatchResult, AccountId}`（`place`/`new`/`best_bid`/`best_ask`）；`engine::money::{Money, MoneyError}`（`apply_rate`/`from_cents`/`cents`/`ZERO`）；`engine::strategy::Rng`（V 演化随机源）；`engine::account::StockCode`（统一 newtype）。
- 涨跌停闭区间 `[down_stop, up_stop]`，边界价合法（涨停价买、跌停价卖接受）。

```rust
//!
//! 设计见 docs/superpowers/specs/2026-06-29-market-design.md。
//! 只做单股状态：涨跌停拒单、last_price 记录、V 演化、日终重置。
//! 不碰 account 结算、不碰全 tick 编排（session/simulator）。

use crate::account::StockCode;
use crate::money::{Money, MoneyError};
use crate::orderbook::OrderError;
use thiserror::Error;

/// market 操作失败。绝不静默吞掉（铁律二）。
#[derive(Debug, Error)]
pub enum MarketError {
    /// 下单价超出涨跌停范围 [down_stop, up_stop]。
    #[error("limit exceeded for {code:?}: price {price:?} not in [{down:?}, {up:?}]")]
    LimitExceeded {
        code: StockCode,
        price: Money,
        down: Money,
        up: Money,
    },
    /// 透传 orderbook 错误（非法价格/数量/tick）。
    #[error(transparent)]
    OrderBook(#[from] OrderError),
    /// 透传 money 错误（涨跌停价/apply_rate 溢出）。
    #[error(transparent)]
    Money(#[from] MoneyError),
    /// V 演化参数非法（volatility/mean_reversion 负或非有限；V 跨零）。
    #[error("invalid v params: {reason}")]
    InvalidVParams { reason: String },
}

/// V（隐藏公允价）演化参数。均值回复几何随机游走。
#[derive(Clone, Debug, serde::Serialize, serde::Deserialize)]
pub struct VParams {
    /// 长期均值（V 向其回复）。
    pub long_run_mean: Money,
    /// 回复速度 α（≥0）。
    pub mean_reversion: f64,
    /// 随机扰动幅度 σ（≥0）。
    pub volatility: f64,
}
```

```rust
use crate::orderbook::OrderBook;

/// 单只股票的市场状态。
pub struct Market {
    code: StockCode,
    book: OrderBook,
    last_price: Money,
    last_close: Money,
    fundamental_value: Money,
    limit_pct: f64,
}

impl Market {
    /// 构造。校验 limit_pct∈(0,1)、initial_price>0、v_initial>0。
    /// last_close = last_price = initial_price（首日无昨收，以开盘价为准）。
    pub fn new(
        code: StockCode,
        initial_price: Money,
        limit_pct: f64,
        v_initial: Money,
        tick: Money,
    ) -> Result<Market, MarketError> {
        if !(limit_pct > 0.0 && limit_pct < 1.0) {
            return Err(MarketError::InvalidVParams {
                reason: format!("limit_pct {limit_pct} not in (0,1)"),
            });
        }
        if initial_price.cents() <= 0 {
            return Err(MarketError::InvalidVParams {
                reason: format!("initial_price {:?} must be > 0", initial_price),
            });
        }
        if v_initial.cents() <= 0 {
            return Err(MarketError::InvalidVParams {
                reason: format!("v_initial {:?} must be > 0", v_initial),
            });
        }
        let book = OrderBook::new(tick)?;
        Ok(Market {
            code,
            book,
            last_price: initial_price,
            last_close: initial_price,
            fundamental_value: v_initial,
            limit_pct,
        })
    }

    /// 涨停价 = last_close × (1 + limit_pct)。
    pub fn up_stop(&self) -> Money {
        self.last_close.apply_rate(1.0 + self.limit_pct).expect("up_stop: last_close finite")
    }

    /// 跌停价 = last_close × (1 - limit_pct)。
    pub fn down_stop(&self) -> Money {
        self.last_close.apply_rate(1.0 - self.limit_pct).expect("down_stop: last_close finite")
    }

    pub fn last_price(&self) -> Money { self.last_price }
    pub fn last_close(&self) -> Money { self.last_close }
    pub fn fundamental_value(&self) -> Money { self.fundamental_value }
    pub fn code(&self) -> &StockCode { &self.code }
}
```
> `apply_rate` 返回 `Result`（limit_pct 已校验有限、last_close 有限，故 `expect` 带说明标注不变量；非业务吞错）。若 clippy 不喜 `expect`，可改为返回 `Result`——但 up_stop/down_stop 语义上恒成功，保持 `expect` + 注释更清晰。

```rust
    /// 下单：涨跌停校验（超限拒单，book 不变）→ 委托 book 撮合 → 末笔成交更新 last_price。
    pub fn place(&mut self, order: Order) -> Result<MatchResult, MarketError> {
        let up = self.up_stop();
        let down = self.down_stop();
        if order.price < down || order.price > up {
            return Err(MarketError::LimitExceeded {
                code: self.code.clone(),
                price: order.price,
                down,
                up,
            });
        }
        let result = self.book.place(order)?;
        // 末笔成交价成为新的 last_price（成交驱动）。
        if let Some(last) = result.trades.last() {
            self.last_price = last.price;
        }
        Ok(result)
    }

    pub fn best_bid(&self) -> Option<Money> { self.book.best_bid() }
    pub fn best_ask(&self) -> Option<Money> { self.book.best_ask() }
    pub fn book(&self) -> &OrderBook { &self.book }
```
> `order.price < down || order.price > up`：闭区间，边界价合法（涨停价买/跌停价卖接受）。`Money` 已 derive Ord 可比较。

## Task 4: `evolve_v`（V 几何均值回复，纯整数银行家舍入）

**Files:**
- Modify: `packages/engine/src/market.rs`
- Modify: `packages/engine/tests/market.rs`

**Interfaces:**
- Produces: `Market::evolve_v(&mut self, params:&VParams, rng:&mut dyn Rng) -> Result<(), MarketError>`。校验 params（α/σ≥0 且有限、mean>0、V>0）；`z=rng.next_f64()*2-1`；`gap=(mean−V)/V`；`drift=α*gap+σ*z`；`multiplier=1+drift`，≤0→Err；`V_new=round_half_to_even_i64(V_cents*multiplier)`，≤0→Err（V 不变）；存入 fundamental_value。模块级私有 `fn round_half_to_even_i64(n:i64,d:u32)->i64`（同 account.rs）+ 本任务另需 `round_half_to_even_f_to_i64(x:f64)->i64`（f64→i64 银行家舍入，用于 V_cents*multiplier）。
- Consumes: `engine::strategy::Rng`（next_f64）、`VParams`、`Money`。

```rust
struct FixedRng(f64);
impl Rng for FixedRng {
    fn next_f64(&mut self) -> f64 { self.0 }
    fn next_range_u32(&mut self, lo: u32, hi: u32) -> u32 { lo }
}

fn evolve_v_mean_reverts_toward_long_run_mean() {
```

```rust
    /// V 几何均值回复一步（种子化）。V_new = V × (1+drift)，drift=α·gap+σ·z。
    /// V 始终为正 Money；multiplier≤0 或 V_new≤0 → Err（V 不变，绝不存非正值）。
    pub fn evolve_v(&mut self, params: &VParams, rng: &mut dyn Rng) -> Result<(), MarketError> {
        // 校验参数
        if !params.mean_reversion.is_finite() || params.mean_reversion < 0.0 {
            return Err(MarketError::InvalidVParams { reason: format!("mean_reversion {} invalid", params.mean_reversion) });
        }
        if !params.volatility.is_finite() || params.volatility < 0.0 {
            return Err(MarketError::InvalidVParams { reason: format!("volatility {} invalid", params.volatility) });
        }
        if params.long_run_mean.cents() <= 0 {
            return Err(MarketError::InvalidVParams { reason: "long_run_mean must be > 0".to_string() });
        }
        let v_cents = self.fundamental_value.cents();
        if v_cents <= 0 {
            return Err(MarketError::InvalidVParams { reason: "current V must be > 0".to_string() });
        }
        let mean_cents = params.long_run_mean.cents() as f64;
        let v_f = v_cents as f64;
        let z = rng.next_f64() * 2.0 - 1.0; // [-1, 1]
        let gap = (mean_cents - v_f) / v_f;
        let drift = params.mean_reversion * gap + params.volatility * z;
        let multiplier = 1.0 + drift;
        if multiplier <= 0.0 {
            return Err(MarketError::InvalidVParams { reason: format!("multiplier {} <= 0 (V would cross zero)", multiplier) });
        }
        let new_v = round_half_to_even_f_to_i64(v_f * multiplier);
        if new_v <= 0 {
            return Err(MarketError::InvalidVParams { reason: format!("evolved V {} <= 0", new_v) });
        }
        self.fundamental_value = Money::from_cents(new_v);
        Ok(())
    }
```

模块级私有 fn（文件末尾）：
```rust
/// f64 → i64 银行家舍入（round-half-to-even）。用于 V 演化的 V_cents×multiplier。
/// 取 floor 与 0.5 比较；半值取偶。负数对称。
fn round_half_to_even_f_to_i64(x: f64) -> i64 {
    let floor = x.floor();
    let diff = x - floor;
    match diff.partial_cmp(&0.5) {
        Some(std::cmp::Ordering::Less) => floor as i64,
        Some(std::cmp::Ordering::Greater) => (floor + 1.0) as i64,
        Some(std::cmp::Ordering::Equal) => {
            if (floor as i64) % 2 == 0 { floor as i64 } else { (floor + 1.0) as i64 }
        }
        None => 0 as i64, // x 为 NaN 等异常（理论上 multiplier 已校验有限，不会达此）
    }
}
```
> `partial_cmp` 防 NaN；`None` 分支理论上不可达（multiplier 已校验 >0、v_f 有限），返回 0 随后 new_v≤0 触发 Err——安全兜底，非静默。
